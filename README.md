#REST API With Flask 

#Login API using Python Flask, JWT, JsonPatching, Opencv

Quick Start Using Pipenv
# Activate venv
pipenv shell

# Install dependencies
pipenv install

# Run Server (http://localhst:5000)
python app.py

#Endpoints

* POST /login
* POST /json_patching?token = secret
* POST /image_thumbnail?token = secret
